# EEPROM-STH

This file contains the momentary used values of the EEPROM from the STH.

## Used Pages

| Page Number | Page Name              |
| ----------- | ---------------------- |
| `0x0`       | System Configuration 0 |
| `0x4`       | Product Data           |
| `0x5`       | Statistic              |
| `0x8`       | Calibration 0          |
